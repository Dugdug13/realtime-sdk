# react-kinetic-ui

A unified React Gesture library with sensor and contactless support, built on top of Framer Motion.

## Features

- **useContactless**: Advanced hand tracking and gesture recognition (MediaPipe).
- **useSensor**: Robust device orientation and motion sensor handling.
- **useSwipe**: High-performance swipe detection.
- **useDrag**: Smooth dragging with boundary support.
- **useTap, usePinch, useRotate, useScroll**: Standard gesture hooks optimized for React.

## Installation

```bash
npm install react-kinetic-ui
```

## Usage

```javascript
import { useContactless, useSensor } from 'react-kinetic-ui';

// ... usage examples
```
